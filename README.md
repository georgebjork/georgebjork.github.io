Since this is currently a mobile app running in the web, the app runs best in a mobile browser. 
`ctrl + shift + i` and then `ctrl + shift + m` is the best way to do this.

When the "Stats" button is pressed, the round summary of match id: 330 is shown. You can view this round here: https://tgin-mvc.azurewebsites.net/match/330
